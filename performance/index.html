显示loader执行时间 #

使用 speed-measure-webpack-plugin可评估每个 loader/plugin 的执行耗时。

npm install -D speed-measure-webpack-plugin
const SpeedMeasurePlugin = require("speed-measure-webpack-plugin");

const smp = new SpeedMeasurePlugin();

const webpackConfig = smp.wrap({
  plugins: [new MyPlugin(), new MyOtherPlugin()],
});

升级项目依赖 #

当前若依使用vue-cli4.x版本，升级为5.x版本, 5.x版本内置webpack5有更好的优化效果 修改package.json文件内容 升级vue-cli

    script: {
       "build:prod": "vue-cli-service build --no-module",
    }

    devDependencies: {
      -"@vue/cli-plugin-babel": "4.4.6",
      -"@vue/cli-plugin-eslint": "4.4.6",
      -"@vue/cli-service": "4.4.6",
      
      +"@vue/cli-plugin-babel": "~5.0.4",
      +"@vue/cli-plugin-eslint": "~5.0.4",
      +"@vue/cli-service": "~5.0.4",
    }

webpack5取消了内置的node包，如path，util 需要安装

npm i path-browserify util

修改 vue.config.js

开启webpack缓存

 configureWebpack: {
    cache: {
      type: "filesystem",
    },
 }

更快的loader: swc-loader #

替换 babel-loader

npm i --save-dev @swc/core swc-loader



 const rule = config.module.rule("js");
    rule.uses.clear();
    rule.exclude
      .add(/(node_modules)/)
      .end()
      .use("swc-loader")
      .loader("swc-loader")
      .options({
        sync: true,
        sourceMaps: true,
        parseMap: true,
        jsc: {
          parser: {
            syntax: "ecmascript",
            jsx: true,
          },
          transform: {
            react: {
              pragma: "vm.$createElement", //"Vue.createElement",
              // pragmaFrag: "Vue.Fragment", //"Vue.Fragment",
              throwIfNamespace: true,
              development: true,
              useBuiltins: true,
            },
          },
        },
      })
      .end();

多进程: thread-loader #

thread-loader 为官方推荐的开启多进程的 loader，可对 babel 解析 AST 时开启多线程处理，提升编译的性能。

module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        use: [
          {
            loader: "thread-loader",
            options: {
              workers: 8,
            },
          },
          "babel-loader",
        ],
      },
    ],
  },
};
