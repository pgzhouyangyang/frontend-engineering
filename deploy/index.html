部署 #

安装 gitlab-runner #

下载

sudo wget -O /usr/local/bin/gitlab-runner https://gitlab-runner-downloads.s3.amazonaws.com/latest/binaries/gitlab-runner-linux-arm

下载完之后，需要给 gitlab-runner 执行权限

sudo chmod +x /usr/local/bin/gitlab-runner

之后创建一个 gitlab-runner 用户，之后使用 CI/CD 时，都是在这个用户下进行的

sudo useradd --comment 'GitLab Runner' --create-home gitlab-runner --shell /bin/bash

创建好了用户之后，接下来安装 gitlab-runner

sudo gitlab-runner install --user=gitlab-runner --working-directory=/home/gitlab-runner

注册 #

首先找到自己的 gitlab 项目下设置，CI/CD，Runner, 找到 url 和 token 执行

gitlab-runner register

运行 #

gitlab-runner start

配置 gitlab-ci.yml #

stages:
  - init
  - lint
  - build
  - deploy
init:
  stage: init
  only:
    - master
  script:
    - npm install
lint:
  stage: lint
  only:
    - master
  script:
    - npm run lint
build:
  stage: build
  only:
    - master
  script:
    - npm run build
deploy:
  stage: deploy
  only:
    - master
  script:
    - rm -rf /home/hl/front/das
    - cp -r dist /home/hl/front/das
